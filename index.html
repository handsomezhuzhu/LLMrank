Ranking Rules

40%

Arena

LMSYS Chatbot Arena ELO score, normalized to 0-100. Real human blind tests where users compare two anonymous models. The most objective third-party ranking.

15%

Ecosystem

Official AI software products: CLI tools, web platforms, desktop/mobile apps, API platforms, developer tools, creative tools, productivity integrations.

15%

Price

Log-compressed output price ($/1M tokens). Cheapest = 100. Log scale compresses high-end gaps — $0.38 vs $2 gets a bigger gap than $12 vs $25.

15%

Multimodal

Model-native capabilities only: Text=25, Thinking=25, Vision/Audio/Video=25, Generation via separate models=25. Not company's other products.

10%

Context Window

Maximum context window size, normalized to 1M tokens = 100 points. Larger context enables processing longer documents and conversations.
